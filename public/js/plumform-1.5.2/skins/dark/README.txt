Installing the "dark" skin:

1. Include dark.css
<link rel="stylesheet" media="all" href="/path/to/skins/dark/dark.css">

2. Include the color you want
<link rel="stylesheet" media="all" href="/path/to/skins/dark/dark-blue.css">

3. Include dark.ie7.css if you have IE7 visitors
<!--[if IE 7]><link rel="stylesheet" media="all" href="/path/to/skins/dark/dark.ie7.css"><![endif]-->
